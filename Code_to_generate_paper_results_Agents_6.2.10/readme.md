This folder stores the code which was used to generate the results of the orginal paper "Agent-based Modelling of Microbialite Formation
through Sedimentation and Precipitation Dynamics". This code was orginally written in Agents.jl 6.2.10 and features some syntax notably AgentBased models are intiated. Hence, this code will crash if one attemots to run it in later versions of
Agents.jl. The code can be run by downgrading to Agent.jl 6.2.10 or if you want to use a more recent version of Agents.jl the code on the main page is identical aside from syntax changes and runs in Agents.jl 7.0.3 and is the recommended way to run the code.


This code also feature some the orginal naming conventiosn where precipiating agents where called MicrobeMat. This reflected an intial conceptualisation of the model but was removed in the most recent version after peer review. For the most up to date convention see the main page.
